# 插件开发指南

Garlic Claw 通过 WebSocket 协议连接外部设备插件。本文档介绍如何使用 Plugin SDK 开发自定义插件。

## 概述

插件通过 WebSocket 连接到服务器（默认 `ws://127.0.0.1:23331`），注册设备能力后，AI 可以在对话中调用这些能力作为工具。

开始前建议先启动开发模式：

```bash
cmd /c other\start-dev.bat
```

## 快速开始

### 1. 创建插件项目

```bash
mkdir packages/plugins/plugin-my-device
cd packages/plugins/plugin-my-device
npm init -y
```

`package.json`:

```json
{
  "name": "@garlic-claw/plugin-my-device",
  "version": "0.1.0",
  "private": true,
  "main": "dist/index.js",
  "scripts": {
    "build": "tsc",
    "start": "node dist/index.js"
  },
  "dependencies": {
    "@garlic-claw/plugin-sdk": "*",
    "@garlic-claw/shared": "*"
  },
  "devDependencies": {
    "typescript": "^5.7.0"
  }
}
```

### 2. 编写插件代码

```typescript
// src/index.ts
import { PluginClient } from "@garlic-claw/plugin-sdk";
import { DeviceType, PluginCapability } from "@garlic-claw/shared";

// 定义设备能力
const capabilities: PluginCapability[] = [
  {
    name: "get_status",
    description: "获取设备状态",
    parameters: {}, // 无参数
  },
  {
    name: "set_brightness",
    description: "设置亮度",
    parameters: {
      level: { type: "number", description: "亮度 0-100", required: true },
    },
  },
];

// 创建插件客户端
const client = new PluginClient({
  name: "my-device-LIVING-ROOM", // 唯一名称
  deviceType: DeviceType.IOT,
  capabilities,
});

// 注册命令处理器
client.onCommand("get_status", async () => {
  return { online: true, brightness: 75 };
});

client.onCommand("set_brightness", async (params) => {
  const level = params.level as number;
  // 实际控制设备逻辑...
  return { success: true, brightness: level };
});

// 启动连接
client.connect();
```

### 3. 配置环境变量

```bash
# 需要有效的 JWT token（通过 /api/auth/login 获取）
export PLUGIN_TOKEN="your-jwt-token"
export WS_URL="ws://127.0.0.1:23331"
```

### 4. 编译运行

```bash
npm run build
npm start
```

## 核心概念

### 插件功能 PluginCapability

每个能力描述一个 AI 可调用的操作：

```typescript
interface PluginCapability {
  name: string; // 唯一能力名，会变成 AI 工具名：pluginName__capabilityName
  description: string; // AI 用这个描述来决定何时调用
  parameters: Record<
    string,
    {
      type: "string" | "number" | "boolean";
      description: string;
      required?: boolean;
    }
  >;
}
```

### 设备类型 DeviceType

```typescript
enum DeviceType {
  PC = "pc",
  MOBILE = "mobile",
  IOT = "iot",
  CUSTOM = "custom",
}
```

### WebSocket 协议

插件通过 WebSocket 与服务器通信，协议消息类型：

| 类型             | 方向        | 说明           |
| ---------------- | ----------- | -------------- |
| `register`       | 插件→服务器 | 注册设备和能力 |
| `register_ack`   | 服务器→插件 | 注册确认       |
| `command`        | 服务器→插件 | AI 调用命令    |
| `command_result` | 插件→服务器 | 命令执行结果   |
| `heartbeat`      | 双向        | 心跳保活       |

SDK 会自动处理注册、心跳和重连逻辑，开发者只需关注命令处理。

## AI 集成

插件注册后，其能力会自动成为 AI 可用工具。例如插件名为 `my-device-LIVING-ROOM`，能力 `set_brightness`，AI 工具名为：

```
my-device-LIVING-ROOM__set_brightness
```

用户可以自然语言说“把客厅灯调到 50%”，AI 会自动识别并调用对应工具。

开发完成后可用下面命令关闭开发模式：

```bash
cmd /c other\stop-dev.bat
```
